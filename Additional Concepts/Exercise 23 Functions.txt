Write a query to display Product Id, Product Description and Category of those products whose category name is Electronics. Do case insensitive comparison.

Table Structure:

ColumnName	DataType	Description
Prodid	NUMBER	Unique ID of the product
Pdesc	VARCHAR2	Description of the product
Price	NUMBER	Price of the product
Category	VARCHAR2	Category of the product
Discount	NUMBER	Discount percentage of the product

Query :

SELECT Prodid, Pdesc, Category
FROM Product
WHERE UPPER(Category) = 'ELECTRONICS';
